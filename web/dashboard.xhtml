Essential Entities Toolbox: Properties & People

Manipulate property-level characteristics such as identification numbers, zoning, and associated persons. Configure dwelling units on multi-unit properties.

Add, adjust, and coordinate person-level information for use by modules across the system.

Code enforcement toolbox:

Access and complete the public-facing code enforcement action request form whose data is to investigate the situation and open a case if deemed appropriate.

Browse code enforcement cases and their phases. Initiate case-related action, such as a monitoring visit. Record case-related interactions, such as phone calls, in-person visits, and emails.

Manipulate settings related to case-level events (e.g. communication events, case origination events, etc.)

View data relating to code non-compliance ciations and their progress through the court system. Create court-related reports and documents.

Municipal Code Tools:

Add code sources to the database, add individual code elements to the sources, and edit existing source-related information.

A code element is associated with a type (e.g. heating/cooling, yard/walkways). Create, edit, and remove types. Once a type is created, code elements can be tagged with this type for easy access during day-to-day work.

Muncipalities have their own code sets. A Code Set is a list of "Enforcable code elements" which are plain code elements with additional information about enforcing that particular element in that particular place. (e.g. "Enforcing 2.3.4b: grass height" in Pitcairn)
